1.	https://philippe-bn.github.io/cart263/projects/project-1/
The Fish is thoughtfully designed. Each fish has variables as properties such as position, velocity, acceleration, size and color which create a dynamic and visually varied system. The behaviors like move(), seek(), and separate() reflects that the creator’s ideas of simulating realistic movement. The seek() allows fish to respond and follow the mouse movement that makes the work interactive while separate() prevents fish from coincide.
However, there is still room for development. The environment is visually simple  that lacks additional elements such as plants and lighting which could make the work more attractive. The fish behavior could also be expanded with smoother avoidance movement when gather in piles.

2. https://hybrid.concordia.ca/a_rrette/projectOne/
This code presents a thoughtfully designed interactive photoshop application that combines creativity with a strong sense of humor. At its core, the program is not just about taking pictures but it creates a small digital environment that gives user special experience to take photos. What makes the project especially interesting is the way it transforms a simple browser camera tool into something that feels playful and polished. 
Rather than placing everything into one long block of commands, the creator breaks the project into meaningful functions such as showPage(), startCamera(), takePhoto(), updateAlbum(), and applyFilter(). This a important programming habits and makes future improvements more manageable.

3.https://lazer6769.github.io/Cart-263/Topics/Project_sand/
From artistic perspective, this code is especially interesting because it turns a simple canvas into a sand box that can pile people's own version of sand. Instead of drawing a image, the program creates a living visual process thaqt alows the user to decide what kind of sand they want to create by providing three options:matrixSlider, extentSlider and gravitySlider. The user does not finish forms onto the screen, instead, they release particles that behave according to gravity and local interaction. This makes the work feel less like traditional sandbox game and more like a digital canvas which the image is constantly being formed and reformed through movement.
